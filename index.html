ToDo List APP

The ToDoList_APP is a React-based project that implements a simple to-do list. It allows users to add, delete, and reorder tasks. The project uses React's useState hook to manage the list of tasks and includes functionalities for handling input changes, adding new tasks, and moving tasks up or down in the list. The app is built using React and Vite, offering a minimal setup for quick development and hot module replacement (HMR). GitHub Repository

Django CRUD Application

The CRUDAPP_DJANGO project is a web application built using Django, a high-level Python web framework. It implements basic CRUD (Create, Read, Update, Delete) operations, allowing users to manage a database of items through a web interface. The project structure includes typical Django components such as models, views, and templates, and it demonstrates how to perform CRUD operations in a scalable and maintainable way. The project also includes HTML files to render the web pages and is designed to be a foundational project for beginners to understand Django.
